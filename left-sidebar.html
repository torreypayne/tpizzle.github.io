Devour.Space

Memorize Everything

Full-stack web application, built with Postgres, Rails, Backbone, TDD done with Rspec. The application is a clone of Anki and Mnemosyne, two prominent spaced-repetition apps that force flashcard items into your long-term memory. The application uses the SuperMemo-2 algorithm to create a dynamic schedule for cards the user will need to review. The application takes in responses and calculates an easiness factor, or "e-factor", which determines when the card is scheduled. With each consecutive repetition, the card is placed further and further down a "forgetting curve", which will eventually put the flashcard item into the user's memory.

Devour.Space under a lens

The back-end is built with User authentication, sessions, and interactive data between users. Users can send messages, share decks, add and create cards, test themselves daily, and view data of their progress with a bar graph. When a User creates a public deck, the deck is added to the Public Directory for others to use. The user can add decks in numerous languages, including Japanese, Russian, and Icelandic. There is also a public leaderboard that is correlated with User contribution via created decks and total reviewed cards.
