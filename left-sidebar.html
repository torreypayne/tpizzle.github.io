Devour.Space

Memorize Everything

Web application built with Ruby on Postgres, Ruby on Rails, Bootstrap/CSS, and Backbone.JS. Partial clone of Anki and Mnemosyne. Based of SuperMemo-2 algorithm by Piotr Wozniak. App tests users on publicly and privately shared decklists, where each card has personalized data based off each user's most recent response.

Devour.Space

Devour.Space is a complete web application. The back-end is built with User authentication, sessions, and interactive data between users. Users can send messages, share decks, add and create cards, test themselves daily, and view data of their progress with a bar graph. When a User creates a public deck, the deck is added to the Public Directory for others to use. The user can add decks in numerous languages, including Japanese, Russian, and Icelandic. There is also a public leaderboard that is correlated with User contribution via created decks and total reviewed cards.
