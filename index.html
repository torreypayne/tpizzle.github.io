Devour.Space

Spaced Repetition Web App, built on Postgres, Ruby on Rails, and Backbone.JS. The application uses the SuperMemo-2 algorithm to create a dynamic schedule which Users will be tested on. Whenever you visit a deck, you will be presented with cards that you are scheduled to review. The schedule will test you on the cards so that over time, all the items will be memorized.

3D Asteroids

3D Asteroids game created using the Famo.us framework. The game takes full use of the Famo.us Physics Engine to incorporate realistic, 3D gravity, force, momentum, and velocity. Additionally, the game provides full rotation and shooting, using the arrow keys, WASD keys, and space bar.
